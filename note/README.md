# Two-page extraction notes / Notas de extração (2 páginas)

Self-contained two-page notes stating and proving the main theorem of the
manuscript (`../paper/main.tex`) — the 2-adic obstruction to modular Lyapunov
functions — together with the robustness corollary and an honest positioning
against the literature. They end with the precise question the notes exist to
ask: *is this positive-integer formulation stated explicitly anywhere?*

Intended use: attach to an email to a researcher, a Math StackExchange
reference-request, or a post on r/Collatz, instead of pointing people at the
full manuscript.

| File | Language |
| --- | --- |
| `nota-obstrucao-lyapunov-pt.tex` / `.pdf` | Português (canônico) |
| `lyapunov-obstruction-en.tex` / `.pdf` | English |

Each `.tex` file is fully self-contained (embedded bibliography, no shared
preamble or `.bib`). Build with `make note` from the repository root, or
`tectonic <file>.tex` here.
